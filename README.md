# 羊毛公益中转站

> 一个面向大家的公益羊毛站合集、投稿入口与风险反馈中心。

访问地址：[https://welfare.sstxww1.asia/](https://welfare.sstxww1.asia/)

## 项目简介

羊毛公益中转站用于整理、分区展示和持续维护公益羊毛站、免费福利入口、实用资源站、OpenAI 兼容中转站和限时活动信息。

这个仓库只放项目介绍、投稿说明和维护文档，不包含站点源码、不存放后台密码、密钥、检测 Key 或任何生产配置。

## 当前功能

- 分类导航：左侧分区导航，默认进入 `白嫖专区`。
- 四个专区：`白嫖专区`、`半付费专区`、`黑榜专区`、`逆向专区`。
- 隐藏真实地址：公开卡片不直接展示站点 URL，只通过“立即访问体验”按钮跳转。
- 风险检测：支持 `/v1/models` 联通检测、模型可信度检测、投毒/隐藏注入检测、SSE 流式稳定性检测。
- 状态展示：站点会显示 `可用`、`疑似降级`、`疑似投毒`、`高风险`、`待复测` 等小状态。
- 自动刷新：页面每 100 秒刷新当前分区的状态摘要。
- 用户举报：支持反馈 `降智/乱答`、`疑似投毒`、`跑路`、`扣费异常`、`无法访问`。
- 贡献检测 Key：用户可以为单个站点贡献检测 Key，Key 只用于该站点风险检测，不公开、不回显。
- 管理维护：管理员可审核投稿、移动分类、置顶、调整排序、处理举报和确认黑榜。
- 防刷加固：投稿、管理员登录、举报、贡献检测 Key 均接入 Cloudflare Turnstile。

## 专区说明

| 专区 | 用途 |
| --- | --- |
| 白嫖专区 | 免费、低门槛、普通用户可直接体验的站点。 |
| 半付费专区 | 有免费额度或基础能力，但高级功能、额度或稳定性可能需要付费。 |
| 黑榜专区 | 已发现明显异常、风险较高、疑似跑路、扣费异常或需谨慎访问的站点。 |
| 逆向专区 | 收录直接可以搞逆向的中转站，不用注入额外提示词。 |

## 风险状态说明

| 状态 | 含义 |
| --- | --- |
| 可用 | 基础联通正常，深度检测未发现明显异常。 |
| 疑似降级 | 可能存在模型错配、假模型、降级模型或响应质量异常。 |
| 疑似投毒 | 检测到隐藏提示词、诱导泄密、异常命令或工具调用注入风险。 |
| 高风险 | 多项检测异常，或举报/检测结果显示风险较高。 |
| 待复测 | 缺少有效检测 Key、近期异常待确认，或需要管理员复核。 |

说明：联通正常不等于安全。站点风险会综合连通性、模型可信度、响应洁净度、SSE 稳定性、用户举报和管理员复核结果判断。

## 如何投稿

推荐直接在网站内点击投稿入口提交，投稿时可以选择专区分类。提交后会进入待审核，不会立刻公开。

也可以通过 GitHub Issues 补充线索：

[提交新站点](https://github.com/sstxww/yangmao-fuli-zhongzhuanzhan/issues/new/choose)

投稿建议提供：

- 站点名称
- 访问链接
- 所属分类
- 简短说明
- 免费额度、限制或付费情况
- 是否需要登录、地区限制或限时活动
- 你认为它值得收录的原因

## 如何反馈风险

如果某个站点出现无法访问、乱答、疑似投毒、跑路或扣费异常，请优先使用网站卡片里的“反馈失效/风险”入口。

反馈不会直接公开刷屏，会进入管理员待处理队列。多人举报或多次检测异常后，站点会进入 `待复测`，管理员确认后才会移动到黑榜。

## 数据与安全说明

- 公开 API 不返回真实站点 URL。
- 公开 API 不返回检测 Key 原文、后台密钥或任何生产配置。
- 检测 Key 验证失败会被丢弃，验证成功后仅加密保存用于风险检测。
- 管理员登录保留限速、锁定、短期会话和统一错误提示，降低爆破风险。
- 升级采用兼容式数据库迁移，保留已有站点、评论、分类和安全记录。

## 维护原则

- 公益优先：优先收录免费、低门槛、对普通用户有实际帮助的站点。
- 人工复核：投稿、举报和黑榜确认都需要管理员处理。
- 风险克制：不收录明显诱导、欺诈、强制付费、恶意跳转或风险过高的内容。
- 持续维护：失效、变质、异常或争议较大的站点会被标注、复测、下架或移入黑榜。
