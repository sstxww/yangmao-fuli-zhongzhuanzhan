## 项目简介

这是我的羊毛公益中转站，用来整理和推荐我亲手检测过的公益羊毛站、免费福利入口、实用资源站和限时活动信息。

我希望它不是一个一次性的链接列表，而是一个可以长期维护、持续变好的公益导航。大家可以访问站点，也可以通过 GitHub Issues 投稿新发现的公益羊毛站。合适的内容会在人工核验后收录。

## 收录原则

- 真实可访问：页面能正常打开，核心功能可用。
- 公益优先：优先收录免费、低门槛、对普通用户有实际帮助的站点。
- 人工检测：已收录内容都经过我亲手访问和基础检测。
- 信息克制：不收录明显诱导、欺诈、强制付费、恶意跳转或风险过高的内容。
- 持续维护：失效、变质或风险升高的站点会被下架或标注。

## 如何投稿

如果你发现了新的公益羊毛站，可以在本仓库提交 Issue。

投稿时建议提供：

- 站点名称
- 访问链接
- 简短说明
- 你认为它值得收录的原因
- 是否需要登录、是否有地区限制、是否限时

投稿入口：[提交新站点](https://github.com/sstxww/welfare-sstxww1-asia/issues/new/choose)
